# 🏠 CampusConnect Home Tab Visual Mockup

## 📋 **Overview**
This document provides visual mockups for the CampusConnect Home tab, showcasing the personalized feed with a 3-tab system: Goods/Services (left), All (center), and Events (right), featuring side pullout panels for advanced filtering.

## 🎨 **Design System**

### **Color Palette**
- **Primary Blue**: #007AFF (buttons, links, focus states)
- **Background**: #F2F2F7 (main feed background)
- **Card Background**: #FFFFFF (post card backgrounds)
- **Text Primary**: #000000 (main text)
- **Text Secondary**: #8E8E93 (secondary text)
- **Border**: #E5E5EA (card borders)
- **Success Green**: #34C759 (offer badges)
- **Warning Orange**: #FF9500 (request badges)
- **Info Blue**: #007AFF (event badges)
- **Pullout Background**: #FFFFFF (filter panel backgrounds)

### **Typography**
- **Primary Font**: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto
- **Heading Sizes**: 20px (post titles), 16px (user names), 14px (body text)
- **Body Text**: 14px (default), 12px (small text)
- **Button Text**: 14px (semibold)

## 🔍 **Tab System & Filtering**

### **Three-Tab Structure**
1. **🏷️ Goods/Services (Left)** - Marketplace posts with Offer/Request options
2. **🏠 All (Center)** - Default view showing all posts
3. **🎉 Events (Right)** - Event-specific posts only

### **Pullout Panel Behavior**
- **Goods/Services Tab**: Left-side pullout with Offer/Request + category filters
- **Events Tab**: Right-side pullout with event-specific category filters
- **All Tab**: No pullout, shows all content

### **Filter Categories**
- **Goods/Services**: Academic, Housing, Transportation, Financial, etc.
- **Events**: Academic, Arts & Culture, Sports & Recreation, Timing, etc.
- **Offer/Request**: Only present in Goods/Services pullout
